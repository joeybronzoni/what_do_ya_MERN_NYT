# 

This is a MERN stack app. 

### Purpose
NYT Article Reader uses the NYT API to populate articles on a webpage according to user query. The user can then save that article to read later.

### Technologies Used
    *React.Js  
    *Node.js/Express  
    *MongoDB/Mongoose ORM  
    *Bootstrap  
    *HTML5
    *NYT API
